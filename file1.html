Introduction

HTML stands for HyperText Markup Language.

It is the standard language used to create and structure web pages. Every website you visit is built using HTML along with CSS and JavaScript.

Simple definition

HTML tells the browser:

  1. What content should appear on the page.
  2. Where the content should appear.
  3. Which content is a heading, paragraph, image, table, button, link, etc.

    4. Think of HTML as the skeleton of a human body.

  4. HTML = Skeleton (structure)
  5. CSS= Clothes and appearance (design)
  6. JavaScript= Brain and muscles (behavior)

    7. Without HTML, a webpage would have no structure.

Meaning of HTML

H = HyperText

HyperText means text that contains links to other pages.

Example

Visit Google

When you click the link, it takes you to another page.

what are the Markup language

Markup languages are systems for annotating text in a way that makes it readable by both humans and computers. They use tags to define elements within a document, specifying how the text should be structured, formatted, and displayed. For example, HTML uses tags like for paragraphs and for headings to structure web content, while XML uses custom tags to define data structures for various applications.
